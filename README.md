# py-Surface-Alchemist-
将三维散点数据，比如温度、压力、密度的数据点表，生成区块匹配度较高的公式，z=f(x,y)

三维散点 z=f(x,y) 函数构建工具 (多区域分析版)

数据格式: 第一行 x,y,z 列名，第二行开始是数值

支持将混乱数据分割成多个区域，每个区域拟合单独的公式
